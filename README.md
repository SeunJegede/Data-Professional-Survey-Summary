# 📊 Survey Data Analysis with Power BI

A data transformation and visualization project using Power BI to uncover insights from survey results.

---

## 🗂️ Data Source

Survey results were collected and stored in a **Microsoft Excel file**, then imported into **Power BI Desktop** for processing and analysis.

---

## 🔧 Data Transformation

Before visualization, the raw data was cleaned in **Power Query**:

- Removed empty rows
- Dropped irrelevant columns and entries
- Fixed inconsistent formatting and values

This ensured the dataset was accurate and ready for analysis.

---

## 📈 Visualizations

Once the data was loaded, interactive visuals were built to highlight s key insights:

- **Average Salary by Job Title** – Compare earnings across different roles
- **Average Participant Age** – Understand the demographic profile of respondents
- **Job Satisfaction Levels** – See how satisfied participants are in their careers

---

## 🛠️ Tools Used

| Tool | Purpose |
|------|---------|
| Microsoft Excel | Raw data storage |
| Power BI Desktop | Data loading, transformation & visualization |
| Power Query | Data cleaning and shaping |
